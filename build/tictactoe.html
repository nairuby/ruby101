Tic Tac Toe

Introduction

Tic Tac Toe is a game often played by young people. It is a solvable game and a good exercise in programming. We will first consider this game without a graphical user interface so that you can try it in the browser if you do not have Ruby installed. Tic Tac Toe is also known as noughts and crosses. It has a 3 x 3 game board and two players. The game starts with an empty board, and players put their pieces on the board in turn. The objective of the game is to get 3 of ones pieces in a line, the first player to do so wins. If neither player does so, then the game ends in a draw. An example game sequence is below.
