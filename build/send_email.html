Sending Email

Ruby can be used to automate your work and make you more efficient. One area that Ruby can be used is in sending email. A nice example of this is in Automate the boring stuff (a nice example of how programs can make you more efficient, unfortunately written for Python, though Ruby can be used for many of the same tasks).

Here, we will have a comma separated value file with fictional members of Nairuby, some of whom will attend an upcoming session of Ruby 101. Our program will send a reminder email to those who are attending the upcoming session of Ruby 101.
