Ruby 101: E-commerce

E-commerce

Ruby on Rails has been popularized because it has enabled many to do e-commerce. E-commerce is the use of the internet for consumer transactions. An early success that has used Ruby is Shopify. An open source Ruby based e-commerce solution is Spree commerce. You can find a demonstration shop here. Spree commerce requires you to run your own server, however you can also try it on your own computer. Follow the instructions to download it here.

Other example open source e-commerce solutions written in Ruby include:

  • Solidus
  • ror_ecommerce
A guide to building an ecommerce application using Ruby on Rails is:
Agile Web Development with Rails 6 by Sam Ruby David Bryant Copeland and Dave Thomas
