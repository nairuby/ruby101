Testing

An important part of developing programs is software testing. Tests try to ensure software quality and eliminate bugs, they are especially useful when there are many software developers. They are also useful when one wants to have secure software and safe software in critical situations.

To get some idea of the testing process, we will try Ruby Koans. This is a set of exercises that can be tried on the command line.

Some other software testing tools include:
  1. Cucumber, with a free online course Behaviour Driven Development with Cucumber
  2. Rspec (actively maintained)
  3. Minitest (actively maintained)
  4. TMF: a minimal testing tool for ruby
  5. Bacon
  6. Capybara for testing web applications
