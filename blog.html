Make a Blog with Ruby

  • GitLab allows you to make a static website for free
  • You can use a static website to host a blog
  • Later, we will also use a static website to host a shop
  • Ruby is used in the static website generator Middleman
  • Ruby will be run automatically for you by a GitLab continuous integration
  • If you do not have an account on GitLab, please create one here
  • Now sign into GitLab here
  • Now fork the following repository
  • Change the content of the first post in file source/2020-05-27-example-article.html.markdown
  • Then go to CI/CD, click on the pipleines option, and Run Pipeline
  • Click on Settings, and pages to see where your page will appear.
  • One can add further styling to the article, but we leave this as a possible project.
