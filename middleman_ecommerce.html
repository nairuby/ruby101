E-commerce with Middleman

  • It is possible to obtain free hosting for a static website. This can make for a cost effective shop. To preserve your brand it is helpful to obtain a domain name. Domain names are rented, so if fees are not paid on a regular basis, someone else can use the domain name. Domain names can be obtained from about 1,000/= per year. Some domain name registrars include:
    • Alibaba
    • Freenom
    • Gandi
    • Hover
    • Kenya Web Experts
    • Namecheap
    • True host
    • Webhost Kenya
    In addition to selling domain names, Freenom
    • also allows you to borrow a domain name for free, though only on certain top level domains.
  • A static website on its own does not allow you to receive payments. To do so, a link to a payment provider will be used.
  • An excellent tutorial on making an e-commerce shop with Middleman has been provided by the payment provider Snipcart, and is available here. Snipcart is great, but at present has a minimum monthly fee of 1000 Kshs or 10 USD. For some E-commerce activities this may be prohibitive. The tutorial here will use Flutterwave, which has no minimum monthly fee, but does keep a percentage of the transaction fee of any sales. Please examine their terms and conditions to determine if it is appropriate for your usecase. Using static websites for e-commerce is new, and has been promoted by Jamstack and associated serverless approaches to the web. As a result, many payment providers require you to use your own server, which is extremely helpful for more complicated e-commerce operations, but can also be more expensive to run, if you do not need it.
